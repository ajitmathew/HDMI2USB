Detailed Description

This module handles the communication of EDID structure with PC and monitor/projector. It contains two sub-modules and one sequaltial logic block. The sequaltial logic block reads the EDID from montor when it detects the HPD from monitor by using edidmaster block. at the same time it pulls the HPD connected with PC HDMI to down, simulating the HDP disconnect. Then PC initiates the i2c communication with FPGA using edidslave block and reads the EDID structure.
